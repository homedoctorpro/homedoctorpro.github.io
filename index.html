How to implement a font (alternate method)

Steps Action
Step 1 Go to https://fonts.google.com/
Step 2 Select all the fonts you want and download them into a folder in your directory
Step 3 Add the following code in this format: @font-face { font-family: 'MyParagraphFont'; src: url('fonts/Roboto.woff2') format('woff2'), url('fonts/Roboto.woff') format('woff'), url('fonts/Roboto.ttf') format('truetype');
